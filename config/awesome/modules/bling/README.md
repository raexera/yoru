## Features

- Layouts
    - mstab (master-slave tab layout)
    - centered
    - vertical
    - horizontal
    - equalarea
- Modules
    - Flash Focus
    - Tabbed container
    - Tiled Wallpaper
    - Wallpaper Easy Setup
    - Window Swallowing
    - Scratchpad
- Signals
    - Playerctl
- Widgets
    - Tag Preview
    - Task Preview

All naming credit goes to [JavaCafe01](https://github.com/JavaCafe01).

### Dependencies

In order to use the `tabbed` modules `pick` function, you need to install `xwininfo`.
